1 Validating the findings about power poses (8 points)

Do power poses really work? Let's analyze the data (keller2017.csv) from:

Keller, V. N., Johnson, D. J., & Harder, J. A. (2017). Meeting your inner super (wo)man: are power poses effective when taught? Comprehensive Results in Social Psychology, 2(1), 106-122.

where

4 experimental conditions were AC (Aware-Constrictive), AE (Aware-Expansive), UC (Unaware-Constrictive), & UE (Unaware-Expansive)

and

3 dependent measures were body expansiveness, performance (in a job interview), & hireability scored by 4 independent raters.

Please construct 3 pandas dataframes df1, and df2, and df3 where

df1, derived from df, has only 13 columns of interest (cond, expansiveness1-4, performance1-4, hire1-4) and no missing values (2 points) ;

df2, derived from df1, has 4 columns that average expansiveness1-4 into "expansiveness", "performance1-4" into "performance", & hire1-4 into "hire" (df1 & df2 should have the same number of rows) (2 points) ;

df3, derived from df2, calculates the means of each dependent variable in each condition (2 points) ;

Finally, run three two-way ANOVAs to see if there is any statistically significant result for each dependent measure (2 point). See this page for an example.

import statsmodels.api as sm
from statsmodels.formula.api import ols

def get_anova_results(df: pd.DataFrame, dep_var: str):
    model = ols(f"{dep_var} ~ C(awareness) + C(body_pos) + C(awareness):C(body_pos)", data=df).fit()
    anova_table = sm.stats.anova_lm(model, typ=2)
    return anova_table
