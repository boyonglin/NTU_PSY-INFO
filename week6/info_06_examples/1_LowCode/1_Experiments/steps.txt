[PsychoPy] (not working for 2023 versions)
1. Export to HTML
2. You need to manually add the "lib" subfolder
3. You need to manually add images into "resources:[]" in the main .js file
4. Switch to the exp folder & "python -m http.server"

(for Pavlovia)
5. "Run on Pavlovia" = git push to Pavlovia (avoid pushing any parent folders)
6. Switch the status of the project from "INACTIVE" to "RUNNING" in Pavlovia

[jsPsych]
1. Download the latest version from https://www.github.com/jspsych/jspsych/releases/latest/download/jspsych.zip

2. You don't need to use a web server to run jsPsych experiments!
